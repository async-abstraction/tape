# Installation

Can be managed using
[yarn](https://yarnpkg.com),
[npm](https://github.com/npm/npm),
or [jspm](https://jspm.io).


### yarn
```terminal
yarn add @aureooms/js-tape
```

### npm
```terminal
npm install @aureooms/js-tape --save
```

### jspm
```terminal
jspm install npm:@aureooms/js-tape
```
